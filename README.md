## 🇧🇷 Português

Sistema simples de gerenciamento de estoque executado no terminal, desenvolvido em Python puro.

### ✨ Funcionalidades

| Opção | Ação                            |
| ----- | ------------------------------- |
| `1`   | Visualizar todo o estoque       |
| `2`   | Adicionar novo produto          |
| `3`   | Remover produto                 |
| `4`   | Buscar produto pelo nome        |
| `5`   | Atualizar quantidade de produto |
| `0`   | Sair do programa                |

### 🚀 Como usar

**Pré-requisitos:** Python 3 instalado.

```bash
# Clone o repositório
git clone https://github.com/santisantos13/python-stock-manager.git

# Entre na pasta
cd python-stock-manager

# Execute o programa
python main.py
```

## 🇺🇸 English

A simple terminal-based inventory management system built with pure Python.

### ✨ Features

| Option | Action                       |
| ------ | ---------------------------- |
| `1`    | View all stock items         |
| `2`    | Add a new product            |
| `3`    | Remove a product             |
| `4`    | Search for a product by name |
| `5`    | Update product quantity      |
| `0`    | Exit the program             |

# Clone the repository
git clone https://github.com/santisantos13/python-stock-manager.git
