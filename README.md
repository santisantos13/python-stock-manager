# Python Stock Manager

## 🇧🇷 Português

Um sistema simples de gerenciamento de estoque desenvolvido em Python no terminal.

## Funcionalidades

- Adicionar produtos
- Atualizar quantidade de produtos
- Verificar se um produto já existe
- Visualizar todo o estoque
- Validação de entradas do usuário

## Tecnologias

- Python 3

## Objetivo

Este projeto foi desenvolvido com o objetivo de praticar:

- Funções
- Estruturas condicionais
- Laços de repetição
- Manipulação de listas
- Validação de dados
- Organização de código
